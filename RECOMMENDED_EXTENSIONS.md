# Recommended VS Code Extensions for SetMyBizz Development

To upgrade your development experience for this Next.js + Tailwind CSS project, install these extensions:

## üé® UI & Styling

1. **Tailwind CSS IntelliSense** (`bradlc.vscode-tailwindcss`)
   - **Why:** Essential for autocomplete class names, hovering to see CSS, and linting.
   - **Status:** *Highly Recommended*

2. **VSCode Icons** (`vscode-icons-team.vscode-icons`) or **Material Icon Theme** (`PKief.material-icon-theme`)
   - **Why:** Makes the file explorer much easier to navigate by adding distinct icons for React, TS, CSS, etc.

## ‚ö° React & Next.js

3. **ES7+ React/Redux/React-Native snippets** (`dsznajder.es7-react-js-snippets`)
   - **Why:** Type `rafce` to create a new component instantly. Saves huge amounts of time.

4. **Auto Rename Tag** (`formulahendry.auto-rename-tag`)
   - **Why:** Renaming an opening `<div>` automatically renames the closing `</div>`.

## üßπ Code Quality & Formatting

5. **Prettier - Code formatter** (`esbenp.prettier-vscode`)
   - **Why:** Automatically formats your code on save (fixes indentation, quotes, etc.).

6. **ESLint** (`dbaeumer.vscode-eslint`)
   - **Why:** Highlights errors and best-practice violations right in your editor.

## üõ†Ô∏è Utilities

7. **GitLens** (`eamodio.gitlens`)
   - **Why:** Shows who changed which line and when. Great for tracking history.

8. **Color Highlight** (`naumovs.color-highlight`)
   - **Why:** Highlights web colors (like #ff0000) with their actual color in the editor.

## üöÄ How to Install

1. Click the **Extensions** icon in the left sidebar (or press `Ctrl+Shift+X`).
